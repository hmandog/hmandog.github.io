Handheld Computer

The handheld computer project is designed to be a powerful, multifunctional calculator with a touchscreen interface. It supports basic calculations, scientific functions, engineering formulas, and more. The project uses the STM32 Nucleo-F411RE development board, a 3.5-inch TFT LCD display, and several other essential components.

Features

  • Basic Mode: Simple arithmetic operations.
  • Scientific Mode: Functions like sin, cos, log, exponents (up to Â±12).
  • Engineering Mode: A menu with predetermined formulas where users provide inputs after selecting the formula.
  • Custom Formula Library: Storage for user-defined formulas.
  • Unit Converter: Conversions between units (length, weight, temperature, etc.).
  • Data Logging: Logs calculations and results to an SD card for future reference.
  • Battery Life Indicator: A display feature showing remaining battery life.
  • Touchscreen Interface: Intuitive interaction using a resistive touchscreen display.

Parts List

  • STM32 Nucleo-F411RE Development Board: Main microcontroller for handling all the operations.
  • 3.5-inch TFT LCD Display (ILI9486): Touchscreen display for user interaction and output.
  • Li-Po Battery (3.7V, 2000mAh): Powers the handheld computer.
  • MT3608 Step-up Boost Converter: Increases voltage to 5V for the components.
  • TP4056 Lithium Battery Charger Module: Handles charging the Li-Po battery.
  • JST XH 2-Pin Connectors: Connects the battery and other components.
  • Micro-USB Cable: For programming and charging.
  • SD Card (8GB): Stores logged data and user-defined formulas.
  • Pin Headers (40P, 2.54mm): Used for connecting components to the STM32 board.
  • LED Indicators: Shows charging status and power on/off state.
  • Heat Sinks (9x9x5mm): Added to prevent overheating of critical components.
  • Resistors and Capacitors: For stabilizing signals and filtering power supply.
